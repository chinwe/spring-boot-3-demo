# Product Overview

This is a Spring Boot demo application that showcases modern Java web development patterns and best practices. The project serves as a reference implementation for building RESTful APIs with Spring Boot 3.x.

## Core Purpose
- Demonstrate Spring Boot 3.5.0 capabilities
- Showcase integration patterns with popular Java libraries
- Provide a foundation for building scalable web applications
- Serve as a learning reference for modern Java development

## Key Features
- RESTful API endpoints with validation
- Object mapping with MapStruct
- Database integration using JOOQ
- Redis caching support
- Comprehensive testing setup with Testcontainers

## Target Use Cases
- API development and testing
- Learning modern Spring Boot patterns
- Template for new Spring Boot projects
- Integration testing demonstrations
